Variables & Types
Day 1 · Fundamentals
What is a Variable?

A variable is a named container that stores a value in memory. Python automatically figures out the type — no need to declare it.

# Assigning variables
name   = "Thamizh"     # str
age    = 21           # int
height = 5.8          # float
active = True         # bool
Data Types
TypeExampleDescription
int10, -5, 0Whole numbers
float3.14, -0.5Decimal numbers
str"hello"Text / characters
boolTrue / FalseBoolean values
Input / Output & Type Casting

All input() values are strings by default. Use type casting to convert them.

name   = input("Enter name: ")          # str
age    = int(input("Enter age: "))       # int
height = float(input("Enter height: "))   # float

print(f"Hi {name}, you are {age} years old and {height}m tall.")
💡 f-strings (f"...") let you embed variables directly inside strings using curly braces {variable}.
Arithmetic Operators
a, b = 10, 3
print(a + b)   # 13  — Addition
print(a - b)   # 7   — Subtraction
print(a * b)   # 30  — Multiplication
print(a / b)   # 3.33 — Division (float)
print(a // b)  # 3   — Floor division
print(a % b)   # 1   — Remainder (modulo)
print(a ** b)  # 1000 — Power

Loops
Day 3 · Iteration
for loop + range()

Repeat code a fixed number of times. range(start, stop, step) generates a sequence of numbers.

for i in range(1, 6):     # 1 2 3 4 5
    print(i)

for i in range(10, 0, -1): # 10 9 8 ... 1
    print(i)

for i in range(2, 11, 2): # 2 4 6 8 10 (evens)
    print(i)
while loop

Repeat as long as a condition is True. Make sure the condition eventually becomes False to avoid infinite loops.

i = 1
while i <= 5:
    print(i)
    i += 1   # must update i to avoid infinite loop

# Infinite loop with break
while True:
    user = int(input("Enter (0 to quit): "))
    if user == 0:
        break
break & continue
# break — exit the loop immediately
for i in range(10):
    if i == 5: break
    print(i)   # prints 0 1 2 3 4

# continue — skip current iteration
for i in range(1, 6):
    if i % 2 == 0: continue
    print(i)   # prints 1 3 5 (skips evens)

Patterns
Day 3 · Nested Loops
How Nested Loops Work

The outer loop controls the row, the inner loop controls the column. Using end=' ' prevents newline after each print; print() at the end of outer loop adds the newline.

for row in range(1, 6):       # outer = rows
    for col in range(1, row+1): # inner = cols
        print("*", end=' ')
    print()   # newline after each row
All 10 Patterns You Practiced
# 1. Full rectangle
for row in range(1,6):
    for col in range(1,6): print("*",end=' ')
    print()

# 2. Right triangle (increasing)
for row in range(1,6):
    for col in range(1, row+1): print("*",end=' ')
    print()

# 3. Inverted triangle (decreasing)
for row in range(5,0,-1):
    for col in range(1, row+1): print("*",end=' ')
    print()

# 4. Diamond (grow then shrink)
for row in range(1,10):
    coltimes = (10-row) if row>5 else row
    for col in range(1, coltimes+1): print("*",end=' ')
    print()

# 5. Increasing number triangle
num = 1
for row in range(1,6):
    for col in range(1, row+1):
        print(num, end=' '); num += 1
    print()
List
Day 4 · Data Structure
What is a List?

An ordered, mutable collection of items. Items can be of any type. Lists support indexing, slicing, and iteration.

nums = [10, 20, 30, 40, 50]

print(nums[0])    # 10  — first
print(nums[-1])   # 50  — last
print(nums[1:3])  # [20, 30] — slice
print(nums[::-1]) # [50,40,30,20,10] — reverse
print(len(nums))  # 5
List Methods
append() insert() remove() pop() sort() count() index() 
nums = [10, 20, 30]
nums.append(40)        # [10, 20, 30, 40]
nums.insert(1, 99)     # [10, 99, 20, 30, 40]
nums.remove(20)        # removes first 20
nums.pop()             # removes last item

nums2 = [5, 1, 9, 2]
nums2.sort()            # [1, 2, 5, 9]

nums3 = [10, 20, 10, 30, 10]
print(nums3.count(10)) # 3
print(nums3.index(30)) # 3 (position of 30)
Nested Lists
matrix = [
    [1, 2],
    [3, 4]
]
print(matrix[0][0])  # 1
print(matrix[1][1])  # 4

for row in matrix:
    for num in row:
        print(num, end=' ')
    print()

Strings
Day 4 · Text Processing
Indexing, Slicing, Reverse

Strings are immutable — you can read characters but cannot change them. They support all list-style indexing and slicing.

word = "PYTHON"
print(word[0])     # P
print(word[-1])    # N
print(word[1:4])   # YTH
print(word[::-1])  # NOHTYP  ← reverse
print(len(word))   # 6
String Methods
upper() lower() replace() split() strip() join() isdigit() isalpha() isupper() 
word = "hello world"
print(word.upper())          # HELLO WORLD
print(word.replace(' ','')) # helloworld
print(word.split())          # ['hello', 'world']

sentence = "I love Python"
words = sentence.split()
print(len(words))            # 3
Palindrome Check
word = "madam"
rev  = word[::-1]
if word == rev:

Anagram Check
word1 = "heart"

Dictionary
Day 5 · Key-Value Pairs
What is a Dictionary?

A dictionary stores data in key: value pairs. Keys are unique identifiers; values can be anything. Use it when you need to label your data.

student = {
    "name": "Ram",
    "age": 20,
    "city": "Chennai"
}

print(student["name"])   # Ram
student["age"] = 21      # update
student["dept"] = "AI"  # add new key
Dictionary Methods & Iteration
# Iterating
for key in student:
    print(key, "->", student[key])

# Better way using .items()
for key, value in student.items():
    print(key, "->", value)

# Check if key exists
print("name" in student)  # True
keys() values() items() get() pop()
Nested Dictionary
employee = {
    "name": "Ram",
    "salary": {
        "jan": 1000,
        "feb": 1200
    }
}
print(employee["salary"]["feb"])   # 1200

employee["salary"]["march"] = 1500  # add

for month, amount in employee["salary"].items():
    print(month, "->", amount)

Set
Day 5 · Unique Collections
What is a Set?

A set is an unordered collection with no duplicates. It's the fastest way to remove duplicates and check membership.

nums = {1, 2, 2, 3, 4, 4}
print(nums)       # {1, 2, 3, 4} — duplicates removed

# Empty set — must use set(), not {}
empty = set()

# Remove duplicates from a list
lst = [1, 2, 2, 3, 3]
unique = list(set(lst))
print(unique)     # [1, 2, 3]
Set Operations
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)  # {1,2,3,4,5}  — Union (all)
print(a & b)  # {3}           — Intersection (common)
print(a - b)  # {1,2}         — Difference (only in a)

a.add(10)     # add element
a.remove(2)   # remove element
⚠️ Sets are unordered — you cannot index them with set[0]. Use a list if order matters.

Tuple
Day 5 · Immutable Sequence
What is a Tuple?

A tuple is like a list but immutable — once created, you cannot change its values. Use it for fixed data that should not change.

nums = (10, 20, 30, 40)

print(nums[0])      # 10
print(nums[-1])     # 40

for i in nums:
    print(i)

# This will RAISE an ERROR:
# nums[0] = 99  ← TypeError: 'tuple' does not support item assignment
Tuple Methods & Conversion
nums = (1, 2, 2, 3, 2)
print(nums.count(2))  # 3  — how many times 2 appears
print(nums.index(3))  # 3  — position of 3

# Conversion
lst = [1, 2, 3]
tup = tuple(lst)       # list → tuple
back = list(tup)       # tuple → list

print(type(tup))       # <class 'tuple'>

marks = [45, 90, 32, 76, 88]

Task 2 — Object Detection Counter
detections = ["person", "car", "person", "dog", "car"]
count_dict = {}

for obj in detections:
    if obj not in count_dict:
        count_dict[obj] = 1
    else:
        count_dict[obj] += 1

print(count_dict)
# {'person': 2, 'car': 2, 'dog': 1}
